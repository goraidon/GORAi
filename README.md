# GORAi プロジェクトコレクション

このリポジトリは、AI・自動化・知識管理などの様々な領域をカバーする**実験的な**プロジェクトコレクションです。各プロジェクトは学習・研究・プロトタイピング目的で作成されており、**完成度や実用性には大きな差があります**。

> ⚠️ **重要**: 多くのプロジェクトは**開発初期段階**または**実験段階**にあり、実用レベルには達していません。

---

## 📁 プロジェクト一覧と開発状況

### 🤖 AIアシスタント・チャットボット

#### [TONARiAi](./TONARiAi/) 
**ChatGPT カスタムGPTsコレクション** | 🔴 **開発初期段階**

業務特化型のChatGPTカスタムボット（GPTs）のアイデア集とリンク集です。

- **現在の状況**: GPTsのアイデアとリンクの整理段階
- **課題**: 
  - 多くのGPTsは構想段階または試作段階
  - 体系的な管理・運用方法が未確立
  - 実用性の検証が不十分
- **技術スタック**: ChatGPT API, カスタムGPTs
- **利用方法**: 各GPTsリンクへの個別アクセス（利用可能性は要確認）

---

### 🔧 開発・自動化ツール

#### [NotionChatgpt](./NotionChatgpt/)
**Notion API プロキシサーバー** | 🟡 **プロトタイプ段階**

NotionのページをChatGPTで検索・取得するための実験的なプロキシサーバーです。

- **現在の状況**: 基本機能の実装済み、限定的な動作確認
- **課題**:
  - エラーハンドリングが不十分
  - 大量データでの動作未検証
  - セキュリティ面の考慮不足
- **技術スタック**: Node.js, Express
- **機能**: Notionページの基本的な検索・取得

#### [ProperNounFinder](./ProperNounFinder/)
**OpenAI API 固有名詞検出・置換ツール** | 🟢 **動作確認済み**

CSVファイル内のテキストから固有名詞を検出し、一般的な表現に置き換える処理ツールです。

- **現在の状況**: 基本機能の動作確認済み、小規模データでのテスト完了
- **制限事項**:
  - 大量データでの性能未検証
  - 精度の継続的改善が必要
- **技術スタック**: Python, OpenAI API (GPT-4o)

#### [UniversalScraper](./UniversalScraper/)
**汎用Webスクレイピングフレームワーク** | 🟡 **プロトタイプ段階**

JSONファイルによる設定ベースのスクレイピングツールの実験的実装です。

- **現在の状況**: 基本的なスクレイピング機能の実装
- **課題**:
  - 複雑なサイトへの対応不足
  - GUI設定機能は構想段階
  - 安定性・信頼性の向上が必要
- **技術スタック**: Python, Selenium, BeautifulSoup

### 📝 コンテンツ・ナレッジベース

#### [KAKiMONO](./KAKiMONO/)
**日次記録・学習ノートアーカイブ** | 🟢 **継続更新中**

日々の学習内容、技術検証、アイデア整理を記録したマークダウンアーカイブです。

- **現在の状況**: 継続的に更新中、個人学習記録として活用
- **内容**: 2024年11月〜2025年2月の学習・思考記録
- **注意**: 個人的な学習記録のため、内容の正確性は保証されません

#### [BookShelf](./BookShelf/)
**デジタル書籍コレクション** | 🟢 **利用可能**

読書記録とおすすめ書籍のリスト管理です。

- **現在の状況**: 基本的な書籍リストとして利用可能
- **内容**: AI・データサイエンス、投資、技術書等40冊以上
- **特徴**: Amazonアソシエイトリンク付き

#### [ArtGallery](./ArtGallery/)
**デジタルアート作品集** | 🔴 **実験段階**

AI生成やデジタル創作による作品の収集実験です。

- **現在の状況**: 作品の収集・整理を実験中
- **課題**:
  - 作品の体系的な分類・管理方法が未確立
  - メタデータや作品説明が不十分
  - ギャラリー機能は構想段階
- **内容**: 40点程度のPNG画像ファイル

#### [Sample](./Sample/)
**テンプレート・サンプル文書** | 🟡 **基本テンプレート**

各種サービス開発で使用するテンプレート文書の収集です。

- **現在の状況**: 基本的なテンプレートの収集段階
- **内容**: プライバシーポリシー、利用規約等のサンプル

## ⚠️ 利用上の重要な注意事項

### 🚨 開発状況について
- **大部分のプロジェクトは実験・学習目的の段階です**
- **本格的な運用や商用利用には適していません**
- **動作しないコードや未完成の機能が含まれます**
- **セキュリティ、エラーハンドリング、性能面で問題があります**

### 💰 コスト・リスクについて
- **API利用による予期しない課金が発生する可能性があります**
- **スクレイピング機能は利用先サイトの規約に注意が必要です**
- **データの損失や破損のリスクがあります**

### 🔧 技術的制限
- **詳細なドキュメントが不足しています**
- **テストケースやバリデーションが不十分です**
- **依存関係の管理が不完全な場合があります**

## 📋 システム要件

### 基本環境
- **Node.js** 14.0以降（NotionChatgpt）
- **Python** 3.7以降（ProperNounFinder, UniversalScraper）
- **Google Chrome** ブラウザ（UniversalScraper）

### 必要なAPIアカウント
- **ChatGPT Plus アカウント**（TONARiAi - 月額20ドル）
- **OpenAI APIアカウント**（ProperNounFinder - 従量課金）
- **Notion APIアカウント**（NotionChatgpt - 基本無料）
